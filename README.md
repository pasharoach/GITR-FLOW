## ⚙️ Настройка

### 1. Получение токена Moodle
1. Войдите в ваш Moodle
2. Перейдите на страницу: `https://your-moodle.com/user/managetoken.php`
3. Создайте новый токен
4. Скопируйте токен в приложение

### 2. Настройка в приложении
1. Откройте **Настройки** (⚙️)
2. Введите **URL вашего Moodle**
3. Вставьте **токен доступа**
4. Выберите **курс** и **задание**
5. Нажмите **Подключить**

## 📖 Использование

### Просмотр студентов
- Список студентов загружается автоматически
- Используйте поиск для быстрого нахождения
- Фильтруйте по статусу: Все, Сдано, Не оценено
- Скачайте все ответы одним кликом с помощью кнопки "📥 Скачать все ответы"

### Оценка работ
1. Выберите студента из списка
2. Просмотрите ответ и файлы
3. Введите оценку (0-100 баллов)
4. Добавьте комментарий (обязательно)
5. Нажмите **Сохранить оценку**

### Массовое скачивание
- **Скачать все ответы** - кнопка в левой панели под информацией о задании для скачивания всех сданных работ
- **Скачать все файлы студента** - для студентов с несколькими файлами доступна кнопка "Скачать все (N)"
- **Умное скачивание** - сначала появляется диалог выбора папки, потом все файлы скачиваются туда автоматически
- **Прямое скачивание** - файлы скачиваются напрямую в выбранную папку через Electron API, без диалогов "Save As"
- **Динамический прогресс-бар** - красивый прогресс-бар показывает прогресс скачивания всех файлов в реальном времени
- **Умные фильтры** - при активном фильтре скачиваются только отфильтрованные работы
- **Расширенная информация** - добавлена строка "Сдано: X/70" в информацию о задании
- Файлы скачиваются по очереди с задержкой для стабильности

### Горячие клавиши
- **A** - переход к предыдущему студенту
- **D** - переход к следующему студенту
- **⌘ + 1-6** - применить шаблон комментария 1-6
- **⌘ + T** - открыть настройки шаблонов

### Шаблоны комментариев
1. Откройте **Меню → Оценивание → Шаблоны комментариев**
2. Введите 6 шаблонов комментариев
3. Используйте **⌘ + 1-6** для быстрого применения

### Работа с файлами
- **Клик** по файлу - открыть предпросмотр
- **Кнопка "Скачать"** - скачать файл
- **Прогресс-бар** - отслеживание загрузки
